# Notification

Reusable native desktop notification helpers extracted from `lepus`.

This module targets `native` and currently ships a Windows-first
implementation. It can be used directly from ordinary MoonBit projects without
depending on `justjavac/lepus_runtime`.

NB: 等功能完善后发布为独立的 `justjavac/notification` 包，当前处于开发阶段。
